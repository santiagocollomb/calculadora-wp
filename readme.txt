=== Nombre del Plugin ===
Versión: 1.0

== Descripción ==

El plugin "Calculadora de Intereses" te permite mostrar una calculadora de intereses con sistema de amortización francés en tu sitio web de WordPress. Esta herramienta es ideal para mostrar a tus usuarios cómo se calculan los intereses y cómo se amortiza el capital en un préstamo.

Características principales:
- Calcula los pagos mensuales y la tabla de amortización basada en el sistema de amortización francés.
- Permite guardar y modificar la tasa de interés en la base de datos desde el panel de administración.
- Opción para mostrar u ocultar la tasa de interés en el formulario de la calculadora.
- Diseño flexible y personalizable que se adapta a tu tema de WordPress.
- Fácil de usar e integrar en cualquier página o entrada utilizando un shortcode.

Utiliza este plugin para proporcionar a tus usuarios una forma interactiva y visual de comprender cómo funcionan los préstamos y los intereses. Puedes utilizarlo en sitios web financieros, de bienes raíces, bancos, o cualquier otro sitio que necesite una calculadora de intereses.

== Instalación ==

1. Sube el directorio `nombre-del-plugin` al directorio `/wp-content/plugins/`.
2. Activa el plugin desde el menú "Plugins" en WordPress.
3. Configura las opciones en el panel de administración.

== Uso ==

Una vez que hayas instalado y activado el plugin "Calculadora de Intereses", sigue estos pasos para usarlo en tu sitio web:

1. Agrega la calculadora de intereses a una página o entrada utilizando el shortcode `[calculadora_intereses]`.

2. Cuando un visitante acceda a la página donde has insertado la calculadora de intereses, verá un formulario donde podrá ingresar el monto total a financiar y la cantidad de cuotas deseadas.

3. Al hacer clic en el botón "Calcular", se utilizará la tasa de interés guardada en la base de datos y se generará una tabla con los detalles de amortización. La tabla mostrará el número de cuota, el monto de la cuota, el monto de interés, la reducción de capital y el capital adeudado.

4. Los visitantes podrán ver la tabla de amortización y comprender cómo se distribuyen los pagos de capital e intereses a lo largo del plazo del préstamo.
